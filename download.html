NPM

Install via Node Package Manager

Yarn

Install via Yarn package manager

Quick Start Guide

1

Install the Library

Choose your preferred installation method from above (NPM, Yarn, or CDN).

2

Import or Include

Import the library in your JavaScript file or include the script tag in your HTML.

import html2canvas from 'html2canvas';
3

Capture HTML Element

Use the html2canvas function to convert any HTML element to canvas.

html2canvas(document.body).then(canvas => { document.body.appendChild(canvas); });
4

Explore & Customize
